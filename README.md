### Сборка и запуск проекта

Перейти в директорию build (при необходимости создать) и выполнить следующие команды:

1. Создать Makefile
```bash
cmake ..
```

2. Собрать исполняемый файл
```bash
make
```

3. Запустить программу
```bash
./app
```

Или же просто
```bash
cmake .. && make && ./app
```
